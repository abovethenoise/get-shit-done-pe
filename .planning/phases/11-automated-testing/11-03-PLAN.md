4. **Test discuss-capability:** Run `node gsd-tools.cjs init discuss-capability --cwd=<dir>`. Verify it loads the capability and returns discussion context.

5. **Test discuss-feature:** Run `node gsd-tools.cjs init discuss-feature --cwd=<dir>`. Verify it loads the feature and returns discussion context.

6. **Test plan-feature:** Run `node gsd-tools.cjs init plan-feature --cwd=<dir>`. Verify planning context is returned.

7. **Test execute-feature:** Run `node gsd-tools.cjs init execute-feature --cwd=<dir>`. Verify execution context is returned.

8. **Test progress:** Run `node gsd-tools.cjs init progress --cwd=<dir>`. Verify progress data reflects the synthetic project state.

9. **Test resume:** Run `node gsd-tools.cjs init resume --cwd=<dir>`. Verify resume context is returned.
